# Third-party provenance

Files under `extensions/` began from the official examples distributed with
`@earendil-works/pi-coding-agent` 0.80.10. Pi is licensed under the MIT License.
The local copies exist so each extension can be reviewed, configured, tested,
and replaced without patching Pi core.

The optional `packages/pi-web-verbs` package follows the typed-action concepts
described by `nlweb-ai/MSR-Web-Verbs` and arXiv:2602.17245. No generated verb or
site-specific Playwright implementation from that repository is copied here.
